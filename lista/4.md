# Zadanie 4: Dodawanie poprawek i walidacja MD5

## Cel zadania
Zadanie polegało na przeanalizowaniu różnic między oryginalną listą słów (`lista.txt` wypakowaną z `lista.zip`) a jej nową, poprawioną wersją (`lista-pop.txt`). Następnie należało wygenerować plik łatki różnicowej, dokonać automatycznej aktualizacji pliku pierwotnego oraz zweryfikować integralność danych za pomocą sum kontrolnych MD5.

## Wykonane kroki i komendy w MSYS2
Z poziomu terminala wykonano następującą sekwencję poleceń:

```bash
# Rozpakowanie archiwum źródłowego
unzip -o lista.zip

# Generowanie łatki różnicowej (Unified diff)
diff -u lista.txt lista-pop.txt > lista.patch

# Automatyczne nałożenie poprawek na plik oryginalny
patch lista.txt < lista.patch

# Weryfikacja kryptograficzna spójności danych
md5sum lista.txt
md5sum lista-pop.txt

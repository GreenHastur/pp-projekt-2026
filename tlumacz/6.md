# Zadanie 6: Przetwarzanie struktur słownikowych JSON5 (Marudny tłumacz)

## Cel zadania
Zadanie polegało na automatyzacji przygotowania plików lokalizacyjnych dla tłumacza. W Etapie 1 należało zdublować linie kluczy w pliku `en-7.2.json5` w taki sposób, aby pierwsza linia stała się zakomentowaną referencją (`//`), a druga pozostała aktywnym kluczem. W Etapie 2, po wydaniu aktualizacji (`en-7.4.json5`), należało wyizolować wyłącznie nowe frazy, które nie występowały w poprzedniej wersji słownika.

## Wykonane kroki i skrypty w MSYS2
Przetwarzanie struktur tekstowych zrealizowano za pomocą poniższych potoków uniksowych:

```bash
# Przygotowanie plików źródłowych
unzip -o tlumacz.zip

# Duplikacja linii z automatycznym komentowaniem referencyjnym
awk '
  /^[[:space:]]*"/ { 
    print "  // " $0; 
    print $0; 
    next 
  } 
  { print }
' en-7.2.json5 > pl-7.2.json5

# Izolacja różnicowa nowych fraz i wykluczenie powtórzeń
grep -F -v -f en-7.2.json5 en-7.4.json5 | grep -E '"[^"]+"' > pl-7.4.json5

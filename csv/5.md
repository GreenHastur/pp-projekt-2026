# Zadanie 5: Konwersja danych (CSV do SQL i SQL do CSV)

## Cel zadania
Zadanie wymagało przeprowadzenia dwukierunkowej migracji danych pomiarowych. Pierwsza część polegała na transformacji pliku tekstowego CSV (`steps-2sql.csv`) do formatu zapytań `INSERT INTO` bazy SQL. Druga część wymagała operacji odwrotnej — ekstrakcji danych z pliku `steps-2csv.sql` do formatu CSV z jednoczesnym przeskalowaniem stempla czasu (Unix Epoch Timestamp) z milisekund do sekund.

## Wykonane kroki i skrypty w MSYS2
W środowisku terminalowym zrealizowano poniższe etapy konwersji:

```bash
# 1. Ekstrakcja danych wejściowych
unzip -o csv.zip

# 2. Transformacja formatu CSV do skryptu SQL (z pominięciem nagłówka)
awk -F';' 'NR>1 {print "INSERT INTO stepsData (time, intensity, steps) VALUES (" $1 ", " $2 ", " $3 ");"}' steps-2sql.csv > steps-2sql.sql

# 3. Ekstrakcja SQL do CSV z redukcją milisekund do sekund
echo "dateTime;steps;synced" > steps-2csv.csv
grep "INSERT INTO" steps-2csv.sql | tr -d "()," | awk '{
    split($0, arr, " ")
    synced = arr[11]; sub(/;/, "", synced)
    steps = arr[10]
    dateTime = arr[9]
    
    # Skalowanie czasu: obcięcie 3 ostatnich znaków (zera milisekund)
    dateTime_sec = substr(dateTime, 1, length(dateTime)-3)
    
    print dateTime_sec ";" steps ";" synced
}' >> steps-2csv.csv

# Zadanie 10: Galeria dla grafika (HTML)

## Cel zadania
Zadanie polegało na automatyzacji procesu budowy sieciowego portfolio (galerii WWW). Należało rozpakować dostarczony szablon starego projektu (`galeria.zip`) i napisać skrypt, który w pętli wygeneruje powtarzalne bloki kodu HTML dla każdego zoptymalizowanego zdjęcia z zadania nr 7 (pliki o wysokości 720px z katalogu `galeria_wyjsciowa`), automatycznie wstrzykując je w miejsce znacznika testowego `blue.png` w pliku `index.html`.

## Rozwiązanie problemów asymetrii layoutu 
Podczas wstępnych testów wygenerowana witryna uległa zniekształceniu (efekt układu *shambled*). Zdjęcia o orientacji pionowej (portretowej) były węższe i wyższe od zdjęć poziomych, co blokowało uniksowy mechanizm opływania bloków CSS (`float: left;`). W efekcie wiersze łamały się nieprawidłowo, generując puste dziury w siatce strony.

Do znacznika `<img>` zastosowanoregułę stylów inline: `style="width: 100%; height: 200px; object-fit: cover;"`. Wymusiło to ujednolicenie wszystkich kafelków do stałej wysokości 200 pikseli, a funkcja `object-fit: cover` automatycznie skadrowała nadmiarowe piksele pionowe bez deformowania i rozciągania obrazów.

## Skrypt automatyzujący w MSYS2
```bash
cd "/c/Users/vared/Desktop/projekt sk"
mkdir -p galeria_www
unzip -o galeria.zip -d galeria_www/

# Kopiowanie zoptymalizowanych zasobów o wysokości 720px z zadania 7
cp galeria_wyjsciowa/*.jpg galeria_www/

cd galeria_www
> dynamiczna_lista.txt

# Generowanie struktury drzewa DOM z poprawką docięcia obrazu (object-fit)
for plik in *.jpg; do
    echo "  <div class=\"responsive\">" >> dynamiczna_lista.txt
    echo "    <div class=\"gallery\">" >> dynamiczna_lista.txt
    echo "      <a target=\"_blank\" href=\"${plik}\">" >> dynamiczna_lista.txt
    echo "        <img src=\"${plik}\" style=\"width: 100%; height: 200px; object-fit: cover;\">" >> dynamiczna_lista.txt
    echo "      </a>" >> dynamiczna_lista.txt
    echo "      <div class=\"desc\">${plik}</div>" >> dynamiczna_lista.txt
    echo "    </div>" >> dynamiczna_lista.txt
    echo "  </div>" >> dynamiczna_lista.txt
done

# Zakresowe wycięcie makiety testowej blue.png i iniekcja nowej bazy kodu za pomocą sed
sed -e '/<div class="responsive">/r dynamiczna_lista.txt' \
    -e '/<div class="responsive">/,/<\/div>[[:space:]]*<\/div>/d' index.html > index_final.html

mv index_final.html index.html
rm -f dynamiczna_lista.txt
cd ..

# Zadanie 7: Fotografik gamoń

## Cel zadania
Zadanie wymagało masowego przetworzenia grafiki rasterowej dostarczonej w dwóch plikach archiwów zbiorczych (`kopie-1.zip` i `kopie-2.zip`). Należało rozpakować strukturę, dokonać ujednolicenia formatów wszystkich obrazów do rozszerzenia `.jpg`, ustawić rozdzielczość DPI na wartość `96x96` oraz przeskalować wysokość każdego pliku do dokładnie `720 pikseli` (zachowując oryginalne proporcje szerokości). Wynik końcowy należało zapisać w osobnym katalogu i skonsolidować do jednego pliku ZIP.

## Analiza problemów i rozwiązanie (Case Study)
Podczas wstępnej analizy zawartości archiwów głównych wykryto specyficzną anomalię struktury — pliki wejściowe zawierały zagnieżdżone podkatalogi oraz dziesiątki mniejszych plików `.zip` podzielonych chronologicznie (tzw. struktura „Incepcji ZIP”). Manualna ekstrakcja była niemożliwa ze względów wydajnościowych. 

Problem rozwiązano poprzez użycie polecenia `find`, które przeszukało strukturę drzewa katalogów i automatycznie wypakowało zawartość wszystkich mniejszych pod-archiwów do jednego, wspólnego folderu roboczego. W skrypcie zastosowano również flagę `MSYS_NO_PATHCONV=1`, co zablokowało automatyczną konwersję ścieżek przez emulator MSYS2 w systemie Windows i zapobiegło błędom biblioteki ImageMagick.

## Skrypt automatyzujący w środowisku MSYS2
```bash
# 1. Instalacja biblioteki ImageMagick oraz narzędzi archiwizacyjnych
pacman -S --noconfirm mingw-w64-x86_64-imagemagick zip unzip
export PATH="/mingw64/bin:$PATH"

# 2. Inicjalizacja struktury katalogów i rozpakowanie archiwów głównych
mkdir -p czyste_zdjecia galeria_wyjsciowa
unzip -o kopie-1.zip -d tymczasowy_1/
unzip -o kopie-2.zip -d tymczasowy_2/

# 3. Spłaszczenie zagnieżdżonej struktury i ekstrakcja zdjęć do jednego folderu
find tymczasowy_1/ tymczasowy_2/ -type f -name "*.zip" -exec unzip -o {} -d czyste_zdjecia/ \;

# 4. Pętla for dokonująca masowej konwersji parametrów binarnej grafiki
for plik in czyste_zdjecia/*; do
    [ -f "$plik" ] || continue
    nazwa="${plik%.*}"
    nazwa=$(basename "$nazwa")
    
    # Konwersja formatu do JPG, ujednolicenie do 96 DPI oraz wymuszenie wysokości 720px
    MSYS_NO_PATHCONV=1 magick "$plik" -density 96 -resample 96 -resize x720 "galeria_wyjsciowa/${nazwa}.jpg"
done

# 5. Konsolidacja przetworzonych plików do finałowego archiwum ZIP i sprzątanie
cd galeria_wyjsciowa
zip -r ../gotowe_prace.zip .
cd ..
rm -rf tymczasowy_1 tymczasowy_2 czyste_zdjecia
